# Project Setup Guide

## Prerequisites

Ensure you have the following installed:

- **Node.js** (>= 16.x)
- **npm** or **yarn**
- **TypeScript** (recommended for strict typing)

## Project Structure

The **Crypto Price Tracker** (Next.js) follows this structure:

```
web-app/
│── README.md              # Project Documentation
│── node_modules/          # Installed Dependencies
│── public/                # Static Assets
│── src/                   # Source Code
│   ├── app/               # Next.js App Router
│   ├── components/        # Reusable UI Components
│   ├── hooks/             # Custom Hooks
│   ├── lib/               # Utility Functions
│   ├── services/          # API Services (e.g., CoinGecko Integration)
│   ├── styles/            # Global Styles
│   ├── types/             # TypeScript Types
│── tests/                 # Testing Files
│── eslint.config.mjs      # ESLint Configuration
│── next-env.d.ts          # Next.js TypeScript Environment
│── next.config.ts         # Next.js Configuration
│── package.json           # Project Dependencies
│── package-lock.json      # Dependency Lock File
│── postcss.config.mjs     # PostCSS Configuration
│── tailwind.config.mjs    # TailwindCSS Configuration
│── tsconfig.json          # TypeScript Configuration
```

## Dependencies

Ensure you install the required dependencies:

```sh
npm install next react react-dom @tanstack/react-query axios dotenv tailwindcss postcss autoprefixer
```

### Dev Dependencies

```sh
npm install -D typescript @types/react @types/node eslint prettier jest @testing-library/react @testing-library/jest-dom
```

## Running the Web App

```sh
cd web-app
npm install
npm run dev
```

## Running Tests

```sh
npm run test
```
